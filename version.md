# 버전 히스토리

## v0.01 (2026-01-28)

### 주요 기능
- React + Tailwind CSS 기반 포트폴리오 웹사이트 초기 구축
- 풀스크린 섹션 기반 스크롤 시스템 구현

### 페이지 구성
1. **Main (Section 01)**
   - 타이핑 애니메이션: "FULL-STACK" → "풀스택개발자 서명세입니다."
   - 프로필 사진 (myong.webp)
   - 연락처 정보 (이메일, 전화번호)
   - 스크롤 인디케이터

2. **About Me (Section 02)**
   - 기본 정보 (이름, 생년월일, 이메일, 전화번호, GitHub)
   - 기술 스택 아이콘 (Frontend, Backend, DevOps, Tools)
   - 학력 정보
   - 자격증 (빅데이터분석기사, SQLD, 정보처리기사)
   - 내부 스크롤 지원

3. **Career (Section 03)**
   - 경력 3곳 카드 형식
   - 접기/펼치기 토글 기능 (화살표 버튼)
   - 회사명, 역할, 기간, 설명 항상 표시
   - 펼쳤을 때: 주요 성과, 기술 태그 표시
   - 내부 스크롤 지원

### 스크롤 동작
- Main에서 스크롤 아래로 → About Me로 전환
- About Me 내부 스크롤 후 끝에 도달하면 → Career로 전환
- About Me 맨 위에서 위로 스크롤 → Main으로 복귀
- Career 맨 위에서 위로 스크롤 → About Me로 복귀
- 모든 스크롤바 숨김 처리

### 디자인
- Apple 디자인 스타일
- 그레이 톤 메인 컬러
- 섹션 간 구분선 (8px 검정색 border)
- 부드러운 애니메이션 (fade-in, scale, typing)
- 완전 반응형 (모바일, 태블릿, 데스크톱)

### 네비게이션
- 햄버거 메뉴 (좌측 상단)
- Scroll to Top 버튼 (우측 하단)
- 섹션별 라벨 (우측 상단)

### 기술 스택
- React 18.3.1
- Vite 6.0.7
- Tailwind CSS 3.4.17
- DevIcon CDN (기술 아이콘)

### 특이사항
- 각 섹션은 100vh 고정 높이
- About Me와 Career는 내부 스크롤 가능
- Career 카드는 기본적으로 접힌 상태로 시작
- 스크롤 감지를 통한 자동 섹션 전환
